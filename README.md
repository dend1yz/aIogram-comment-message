# Aiogram Comment Message Send
***
```
Это простенький код, который позволит уведомлять ваших участников о том, что у вас есть чат, в котором они могут общаться

Вы так же можете использовать этого бота для других предназначений, изменяя инлайн кнопки, текст, и картинку.
```
***
```
Установить его очень просто, достаточно скачать ZIP архив, и в bot.py и handlers.py поменять значения на ваши. И бот готов!
```
